This Application accesses an API that have a minimum polulation and a maximum population. You can see the population of the city, the city ID, the region code and the country code. 
After you enter the min and max population you can then press on the search for cities button and then it will display a ton of information for different cities 
The API endpoints that is accessed are min populationa and max population for "GET Cities" and radius and cityid for "GET Cities Near City"
